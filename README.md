# Chaos Explorer

A cleaner way to explore the ProjectDiscovery Chaos dataset. Useful for Bug Bountry hunters for finding programs. 

## Why it Better ?

- Search programs
- Filter by platform
- Filter by bounty status
- Sort by subdomain count
- Mobile-friendly interface
- Fast local caching
